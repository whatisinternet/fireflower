## Presentation:

### I'm new here and on mac/*nix
- This will do the compile step too

```bash
./bin/setup
```

### To compile the presentation:

```bash
yarn compile
```

### To compile the presentation for gh-pages:

```bash
./bin/docs
```

### To view:

```bash
open ./dist/index.html
```

### To clean up:

```bash
yarn gclean
```
